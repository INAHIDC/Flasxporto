Flask portfolio at nchy.dev

About 
Built using Flask and Bootstrap, it showcases my projects, skills, and experience. live at nchy.dev.
AWS EC2 for hosting because of its reliability and scalability. For security and performance, I've configured Caddy to handle HTTPS and serve as a reverse proxy using Port 6060.

Clone this repo to your local machine.
Install dependencies - pip install -r requirements.txt.
Configure any environment variables.
run python app.py.
Open your browser at http://localhost:6060 
~change caddy files port number for hosting.
